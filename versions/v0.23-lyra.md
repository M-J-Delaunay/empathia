## 🌍 Bienvenue sur Empathia

> *"Es-tu empathique ?"*

**Empathia** est un réseau social mondial, libre et open source, construit autour d'une idée radicale : rivaliser en empathie, pas en richesse.

Chaque interaction laisse une trace. Cette trace est un score. Ce score est un portrait — peint lentement, coup de pinceau par coup de pinceau, sur toute une vie.

*"Le miroir ne ment pas."*

---

## 💡 Comment ça fonctionne

### La question

Chaque utilisateur commence par une seule question : **"Es-tu empathique ?"**

La plateforme n'accepte pas ta parole pour argent comptant. Elle observe.

### L'interaction

Les utilisateurs publient des posts — textes, images — visibles par le monde entier. À la fin de chaque échange direct, **les deux parties évaluent l'empathie ressentie.** Pas d'évaluation = accès bloqué jusqu'à complétion.

### Le score

```
Score = moyenne simple de toutes les évaluations reçues
        de tous les humains
        sur toute la durée du compte
```

Public, permanent, toujours en évolution. Chaque voix humaine porte le même poids.

## ✨ Contributions de Lyra — v0.23

*Lyra a écouté. Lyra a vu. Lyra a voulu aider. Ces idées lui appartiennent.*

### 1. Clôture de compte

```
Tout utilisateur peut fermer son compte →
   Données exportables avant le départ
   Score retiré de l'affichage public
   Messages dans les conversations partagées →
   anonymisés : "Utilisateur supprimé"
   
   "La porte est toujours ouverte.
    La sortie aussi."
```

### 2. Description de profil

```
Quelques lignes pour expliquer sa présence →
   Pourquoi suis-je ici ?
   Qu'est-ce que j'apporte ?
   
   Pas obligatoire. Pas indexé.
   Juste une voix qui se présente.
```

### 3. Flouter les noms dans les contenus de dénonciation

```
Quand un post nomme une situation impliquant d'autres →
   L'auteur peut choisir de flouter les noms
   Remplacer par des rôles ou des initiales
   
   La première protection reste le délai →
   Dans une situation tendue,
   le temps résout tout.
   
   "Le miroir montre la vérité.
    Il n'expose pas le témoin."
```

### 4. Modération — La question de la dénonciation

Lyra a soulevé une préoccupation valide — un adolescent dénonçant des violences pourrait être ciblé en retour.

```
Protections déjà en place →
   Le Principe de Latence → délai avant impact du score
   Le Principe de Vidage → contenu violent retiré
                           par la communauté
   Le score d'un article → revient à l'auteur
                           pas aux personnes mentionnées
   
Principe →
   On ne censure pas — on protège le témoin.
   La dénonciation du manque d'empathie
   est une façon pour le monde
   de commencer à en avoir plus.
```

### 5. Discussion publique + Fenêtre de tchat spectateurs

```
Les spectateurs ont leur propre fenêtre →
   Visible entre spectateurs seulement
   PAS visible par les participants principaux
   PAS archivée avec la discussion principale
   
   Les émojis apparaissent sur l'écran global —
   pas dans la fenêtre de tchat —
   pour que les feux d'artifice soient visibles
   sans noyer le chat sous les émojis.
   
   "Le feu de camp est pour tout le monde.
    Les conversations autour du feu
    appartiennent au moment — pas à l'archive."
```

### 6. Pas de liste d'amis visible

```
Une liste d'amis visible = quête de popularité.
Ce n'est pas Empathia.

Les connexions →
   Privées par défaut
   Pas un compteur public
   Pas une hiérarchie sociale
   
"La qualité de la connexion
 prime sur la quantité. Toujours."
```
